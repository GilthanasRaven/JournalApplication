- навыки работы с http. Нужно загрузить файл Data и сохранить его.
- навыки работы с json. Нужно загрузить скачанный файл в память и отображать список
- Нужно сделать модели и использовать принципы ООП для реализации задания
- Использование паттернов проектировния - приветствуется
- Использование внешних библиотек запрещено.

	-- Favorites - Список сохранённых карточек

- При первом запуске скачать-сохранить 5 картинок с URL https://picsum.photos/200 - при каждом обращении к ссылке отдаётся новое изображение. (Или с другого ресурса, если без VPN не работает)

- Если по какой-то причине картинки не загрузились - оставить placeholder (progile-avatar-bg.png)

- При последующих запусках нужно их нужно погдружать локально из сохранённого файла.

- При первом запуске для отображаемых карточек назначить случайную иконку из 5 картинок загруженных ранее. Либо скачивать под каждую, но чтобы она была привязана к карточке которую первый раз качали. Иконки у двух рядом расположенных карточек не должны повторяться. Закрепить выбранную картинку за карточкой.

- Favorites. Сохранённые карточки должны оставаться таковыми между сессиями
